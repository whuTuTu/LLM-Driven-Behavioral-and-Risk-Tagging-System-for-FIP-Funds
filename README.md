# LLM-Driven-Behavioral-and-Risk-Tagging-System-for-FIP-Funds

基于大语言模型的"固收+"基金经理风格画像与标签构建系统

## 项目简介

本项目利用大语言模型对二级债基和转债基金过去三年的定期报告进行深度文本挖掘，构建了一套涵盖"收益风险、操作风格、个性特征"三大维度的客观、可量化标签体系与多维画像。

## 核心功能

- **数据提取**：从 PDF 定期报告中精准提取核心持仓数据与主观文本段落
- **标签生成**：
  - 收益风险标签：收益率、超额收益、夏普比率、卡玛比率等
  - 操作风格标签：左侧布局/右侧跟随/均衡配置
  - 个性特征标签：信用下沉、权益风格、回撤控制、利率波段
- **时序分析**：通过时序多数投票机制，提取穿越周期的长效风格

## 技术栈

- **语言模型**：DeepSeek-V3.2
- **数据来源**：ifind、PDF 定期报告
- **提取方式**：正则表达式 + PDF 解析引擎

## 成本估算

单个基金分析（约12个报告）成本约 0.055-0.11 元

## 标签体系

### 收益风险标签
近一/二/三年收益率、超额收益、标准差、最大回撤、夏普比率、卡玛比率、索提诺比率、胜率

### 操作风格标签
强左侧特征 / 中等左侧特征 / 均衡特征 / 中等右侧特征 / 强右侧特征

### 个性特征标签
- 信用下沉：积极信用下沉 / 适度信用下沉 / 高等级为主
- 权益风格：价值风格 / 成长风格 / 平衡风格
- 回撤控制：极强 / 较强 / 中等 / 较弱
- 利率波段：积极波段操作 / 适度波段操作 / 持有到期为主
