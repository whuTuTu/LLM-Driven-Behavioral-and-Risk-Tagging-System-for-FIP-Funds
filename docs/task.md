1.基于大语言模型的“固收+“基金经理风格画像与标签构建
【题目描述】

1)利用大语言模型对“固收+“基金(二级债基、转债基金)过去3年的定期报告(如一季报、中报、三季报、年报等)进行分析，提取基金经理在宏观研判、大类资产观点等方面的信息以及基金经理的履历、历史业绩、基金持仓信息等信息。

2)在数据分析的基础上，形成一套客观、可量化的基金经理标签体系，该体系应涵盖但不限于以下维度:收益风险标签(高波、中波、低波);操作风格标签(左侧布局、右侧跟随等);个性特征标签(信用下沉、权益成长风格、回撤控制严格、利率波段收益高等)等。


【补充说明】

1、二级债基和转债基金这两类固收+基金在资产配置的限制、收益风险特征等多方面有不同的特性，在研究时建议分开讨论和分析。
2、为使得分析的固收+基金样本更具有代表性，建议对其存续时间、规模做出筛选，如限定:(1)成立时间早于2023年;(2)最新规模不低于2亿且近三年平均规模不低于2亿;(3)目前仍存续。

3、该研究需要的底层数据包括但不限于基金的定期报告(如一季报、中报、三季报、年报等)和基金的净值(复权单位净值)等。相关数据可以从巨潮资讯网(htp:/Wwww.cninfo.com.cn)、中国证监会基金信息披露网站(http://eid.csrc.gov.cn)等官方渠道或者wind、同花顺等第三方数据平台获取。在着手分析之前需要参赛者自行对提供的底层数据进行清洗等操作。
