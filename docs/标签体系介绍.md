# 基金经理风格画像标签体系介绍

## 一、体系概述

本标签体系旨在通过多维度、多层次的标签化方法，全面刻画基金经理的投资风格、操作特征和个性特点。体系分为三大核心模块：

1. **收益风险标签** - 基于历史业绩数据，量化评估基金经理的收益能力和风险控制水平
2. **操作风格标签** - 基于持仓变化和投资策略，分析基金经理的操作习惯和交易特征
3. **个性特征标签** - 基于持仓结构和宏观观点，提炼基金经理的投资理念和风险偏好

---

## 二、标签体系架构

### 2.1 收益风险标签

**目标**：量化评估基金经理的收益能力和风险控制水平

**数据来源**：基金净值数据（从`data/fund_nav/`目录读取）

| 标签名称 | 计算方法 | 取值范围 | 说明 |
|---------|---------|---------|------|
| 近一年收益率 | 净值数据计算 | 数值% | 近一年累计收益率 |
| 近两年收益率 | 净值数据计算 | 数值% | 近两年累计收益率 |
| 近三年收益率 | 净值数据计算 | 数值% | 近三年累计收益率 |
| 近一年标准差 | 净值数据计算 | 数值% | 近一年收益率标准差 |
| 近三年标准差 | 净值数据计算 | 数值% | 近三年收益率标准差 |
| 近一年超额收益 | 净值数据计算 | 数值% | 近一年相对基准超额收益 |
| 近两年超额收益 | 净值数据计算 | 数值% | 近两年相对基准超额收益 |
| 近三年超额收益 | 净值数据计算 | 数值% | 近三年相对基准超额收益 |
| 最大回撤 | 净值数据计算 | 数值% | 近三年历史最大回撤幅度 |
| 夏普比率 | 净值数据计算 | 数值 | 近三年风险调整后收益 |
| 卡玛比率 | 净值数据计算 | 数值 | 近三年年化收益/最大回撤 |
| 索提诺比率 | 净值数据计算 | 数值 | 近三年下行风险调整收益 |
| 波动水平 | 净值数据计算 | 低波/中波/高波/未知 | 基于近三年标准差分类 |
| 胜率 | 净值数据计算 | 数值% | 近三年月度胜率 |

**计算说明**：
- 所有指标均基于基金净值数据计算
- 使用近三年数据计算风险指标（最大回撤、夏普比率、卡玛比率、索提诺比率）
- 波动水平分类标准：
  - 低波：年化标准差 < 3%
  - 中波：3% ≤ 年化标准差 < 8%
  - 高波：年化标准差 ≥ 8%
- 超额收益基于净值文件中的基准数据计算
- 无风险利率默认为2.5%（用于夏普比率和索提诺比率计算）

**应用场景**：
- 基金筛选：根据收益风险特征筛选符合需求的基金
- 风险评估：评估基金经理的风险控制能力
- 业绩归因：分析收益来源和风险暴露

### 2.2 操作风格标签

**目标**：分析基金经理的操作习惯和交易特征

| 标签名称 | 数据来源 | 计算方法 | 取值范围 | 说明 |
|---------|---------|---------|---------|------|
| 投资风格 | 宏观观点+持仓数据 | 多维度综合评分 | 左侧布局/右侧跟随/均衡配置 | 仓位调整时机选择 |

**计算方法详解**：

#### 投资风格标签生成逻辑

投资风格标签采用**多因子打分/投票制**，通过两个维度综合判断基金经理的投资风格：

```
最终得分 = 文本维度得分 × 40% + 持仓维度得分 × 60%
```

**知行合一判断机制**：
- **知行合一**：文本维度和持仓维度方向一致 → 强左侧/强右侧
- **风格漂移**：文本维度和持仓维度方向不一致 → 风格漂移预警

**判定规则**：
- 最终得分 ≤ -0.3 → **左侧布局（逆向投资）**
- 最终得分 ≥ 0.3 → **右侧跟随（趋势投资）**
- -0.3 < 最终得分 < 0.3 → **均衡配置（择时不明显）**

**特殊标签**：
- **强左侧布局（知行合一）**：文本说"逢低布局"，持仓动作"大跌加仓"
- **强右侧跟随（知行合一）**：文本说"顺势而为"，持仓动作"追随趋势"
- **左侧布局（风格漂移预警）**：文本说"左侧潜伏"，但持仓动作"高位加仓"
- **右侧跟随（风格漂移预警）**：文本说"右侧跟随"，但持仓动作"低位减仓"
- **均衡配置（风格漂移预警）**：文本和持仓方向明显不一致

---

#### 维度一：文本维度分析（权重40%）

**数据来源**：基金投资策略和运作分析文本（宏观观点数据）

**分析方法**：
1. **主要方法**：使用LLM（DeepSeek）进行深度语义分析
   - 提取文本中关于投资策略、仓位调整、市场判断的描述
   - 分析基金经理的投资理念和操作风格
   - 评分范围：-1到1（-1为绝对左侧，1为绝对右侧）
   - Prompt示例：
     ```
     请分析以下基金经理的投资策略和运作分析，判断其投资风格是"左侧布局"还是"右侧跟随"。
     
     左侧布局特征：逆向投资、逢低买入、提前布局、底部区间、低估值、防守反击、中低价转债、跌出价值
     右侧跟随特征：顺势而为、追逐趋势、趋势确立、景气度上升、动能、突破、确认拐点
     
     请给出一个得分（-1到1之间）：
     - -1：绝对左侧布局（逆向投资）
     - 0：均衡配置
     - 1：绝对右侧跟随（趋势投资）
     ```

2. **备用方案**：当LLM不可用时，使用关键词匹配
   - **左侧关键词**：逆向、左侧、底部、低估值、防守、中低价、跌出价值、提前布局、逢低、安全边际、潜伏
   - **右侧关键词**：顺势、右侧、趋势、景气度、动能、突破、确认、跟随、追涨
   - **计算公式**：`得分 = (右侧关键词数 - 左侧关键词数) / 总关键词数`

**评分规则**：

| 特征类型 | 得分范围 | 说明 |
|---------|---------|------|
| 强左侧特征 | -1.0 到 -0.6 | 明确的逆向投资特征，如"越跌越买"、"左侧布局" |
| 中等左侧特征 | -0.6 到 -0.3 | 有左侧倾向，但不够强烈 |
| 均衡特征 | -0.3 到 0.3 | 无明显左右侧倾向，保持均衡配置 |
| 中等右侧特征 | 0.3 到 0.6 | 有右侧倾向，但不够强烈 |
| 强右侧特征 | 0.6 到 1.0 | 明确的趋势投资特征，如"顺势而为"、"右侧跟随" |

#### 维度二：持仓维度分析（权重60%）

**数据来源**：
- 基金资产组合情况（股票占比 + 可转债占比）
- 上证综合指数历史数据
- 历史持仓数据（从提取数据文件中读取）

**研究样本**：二级债基和转债基金

**核心逻辑**：通过**仓位变化与市场走势的关系**判断投资风格

**分析步骤**：

**步骤1：获取上一季度市场走势**
```
上一季度指数涨跌幅 = (季度末价格 - 季度初价格) / 季度初价格 × 100%
```

**步骤2：计算当前风险资产占比**
```
风险资产占比 = 股票占比 + 可转债占比

其中，股票占比优先从"按行业分类的境内股票投资组合"的合计中提取，
如果提取失败，则从"报告期末基金资产组合情况"的"权益投资"字段提取。
```

**步骤3：获取上一季度风险资产占比**
```
从历史持仓数据文件中读取上一季度的风险资产占比
```

**步骤4：计算仓位变化**
```
仓位变化 = 当前风险资产占比 - 上一季度风险资产占比
```

**步骤5：根据市场走势和仓位变化判断投资风格**

**核心逻辑**：考虑市场涨跌对仓位的被动影响，区分主动操作和被动漂移

| 市场环境 | 仓位变化 | 得分 | 风格倾向 | 逻辑说明（核心依据） |
|---------|---------|------|---------|---------------------|
| 大幅下跌（<-5%） | > 1% (加仓) | -0.6 | 强左侧 | 克服净值缩水被动下降，不仅补仓还大幅加码，绝对的越跌越买 |
| 大幅下跌（<-5%） | -1% ~ +1% (维持) | -0.3 | 偏左侧 | 市场大跌本会导致仓位被动缩水，能维持比例不变，说明进行了逆势补仓（再平衡） |
| 大幅下跌（<-5%） | -3% ~ -1% (微降) | 0.0 | 均衡 | 属于市场下跌导致的自然缩水（被动漂移），基金经理基本没动 |
| 大幅下跌（<-5%） | < -3% (减仓) | +0.6 | 强右侧 | 仓位下降幅度远超市场跌幅，说明在主动割肉止损、右侧规避趋势 |
| 大幅上涨（>5%） | > 3% (加仓) | +0.6 | 强右侧 | 仓位上升幅度远超市场涨幅，说明在主动追高、顺势右侧加码 |
| 大幅上涨（>5%） | +1% ~ +3% (微升) | 0.0 | 均衡 | 属于市场上涨导致的自然膨胀（被动漂移），基金经理基本没动 |
| 大幅上涨（>5%） | -1% ~ +1% (维持) | -0.3 | 偏左侧 | 市场大涨本会导致仓位上升，能维持比例不变，说明在逆势卖出止盈（再平衡） |
| 大幅上涨（>5%） | < -1% (减仓) | -0.6 | 强左侧 | 市场大涨时仓位反而大幅下降，说明在强力落袋为安、绝对的左侧撤退 |
| 震荡（-5% ~ 5%） | 任何变化 | 0.0 | 均衡 | 震荡市缺乏明确趋势，此时加减仓更多是底层的结构性调仓，不构成判断左右侧的强信号 |

**逻辑依据**：
- **左侧布局**：在市场下跌时加仓（越跌越买），或在市场上涨时减仓（左侧止盈）
- **右侧跟随**：在市场上涨时加仓（顺势加仓），或在市场下跌时减仓（右侧止损）
- **均衡配置**：仓位变化属于市场涨跌导致的被动漂移，基金经理基本没动
- **被动漂移**：市场涨跌会导致仓位被动变化，需要区分主动操作和被动影响

**降级方案**：
当无法获取指数数据或历史持仓数据时，返回None，仅使用文本维度得分。

**数据来源优先级**：
1. 股票占比：优先从"按行业分类的境内股票投资组合"的合计中提取
2. 可转债占比：从"按债券品种分类的债券投资组合"中提取"可转债（可交换债）"字段
3. 历史持仓：从`output/fund_analysis/extracted_data/`目录下的历史报告文件中读取

---

#### 标签含义详解

**左侧布局（逆向投资）**：
- 提前预判市场走势，在市场底部加仓，顶部减仓
- 核心特征：逆向投资、逢低买入、提前布局、底部区间、低估值、防守反击
- 行为模式：在市场下跌时逆势加仓，"越跌越买"；在市场上涨时提前减仓，"越涨越卖"
- 适用场景：震荡市或熊市环境，价值投资导向的基金

**右侧跟随（趋势投资）**：
- 跟随市场趋势，在趋势确认后调整仓位
- 核心特征：顺势而为、追逐趋势、趋势确立、景气度上升、动能、突破
- 行为模式：在趋势确认后跟进，顺势操作；在上涨趋势中加仓，下跌趋势中减仓
- 适用场景：牛市或趋势明显的市场环境，成长投资导向的基金

**均衡配置（择时不明显）**：
- 保持相对稳定的仓位，较少进行主动择时操作
- 核心特征：无明显择时倾向，保持稳定仓位，注重资产配置
- 行为模式：维持相对稳定的权益仓位，较少根据市场波动调整仓位
- 适用场景：震荡市或方向不明确的市场，追求稳健收益的基金

---

**应用场景**：
- 操作风格识别：识别基金经理的操作习惯和择时倾向
- 基金筛选与推荐：根据投资者风险偏好和市场环境推荐相应风格的基金
- 组合构建：在投资组合中配置不同风格的基金，降低单一风格风险
- 风险管理：监测基金投资风格是否发生漂移
- 知行合一检验：验证基金经理的实际操作与宣称的投资理念是否一致

### 2.3 个性特征标签

**目标**：提炼基金经理的投资理念和风险偏好

| 标签名称 | 数据来源 | 计算方法 | 取值范围 | 说明 |
|---------|---------|---------|---------|------|
| 信用下沉 | 宏观观点+持仓数据 | LLM综合分析 | 积极信用下沉/适度信用下沉/高等级为主（信用中性） | 信用债投资偏好 |
| 权益风格 | 宏观观点+持仓数据 | LLM综合分析 | 价值风格/成长风格/均衡风格/无权益持仓 | 权益投资风格 |
| 回撤控制 | 宏观观点+净值数据 | LLM综合分析 | 回撤控制严格/风控执行漂移/适度风控/风控较弱 | 回撤控制严格度 |
| 利率波段 | 宏观观点+持仓数据 | LLM综合分析 | 利率波段交易型/配置持有型/适度参与/不参与 | 利率波段操作能力 |

**核心特点**：
- ✅ **所有个性特征标签均使用LLM进行深度语义分析**
- ✅ **每个标签都有降级方案**：当LLM不可用时，自动降级为关键词匹配或简单判断
- ✅ **多维度综合判断**：结合文本维度和持仓/数据维度
- ✅ **基金类型增强**：支持根据基金类型注入上下文信息

**基金类型区分机制**：

本系统支持根据不同基金类型（二级债基、可转债基金等）进行差异化分析，在LLM Prompt中注入基金类型特性信息，提高分析精准度。

**二级债基 vs 可转债基金对比**：

| 维度 | 二级债基 | 可转债基金 |
|------|---------|-----------|
| 资产配置 | 股票≤20%，可转债0%-10%，债券70%-90% | 可转债≥60%，股票0%-5%，债券0%-30% |
| 波动水平 | 中低波 | 中高波 |
| 目标回撤 | 2%-5% | 5%-15% |
| 主要收益来源 | 债券票息 + 利率波段 + 股票增强 | 转债股性收益 + 转债债性保护 |
| 主要风险来源 | 利率风险 + 信用风险 + 权益风险(次要) | 正股下跌风险 + 转债估值风险 |
| 操作重点 | 股票仓位择时、转债配置、债券久期管理 | 转债仓位管理、转债估值分析、偏股型/偏债型配置 |
| 分析指标 | 权益仓位变化、转债仓位变化、债券久期 | 转债仓位、平均转股溢价率、偏股型转债占比 |

**计算方法详解**：

#### 1. 信用下沉分析

**分析方法**：LLM综合分析（文本维度 + 持仓维度）

**文本维度**：
- 使用LLM解析投资策略文本，判断是否提及信用下沉策略
- Prompt示例：
  ```
  请分析以下基金经理的投资策略，判断其信用下沉程度。

  【基金类型特性】
  {基金类型上下文}

  投资策略文本：{strategy_text}

  请判断：
  1. 是否明确提及信用下沉策略？
  2. 信用下沉的激进程度如何？
  ```

**持仓维度**：
- 使用LLM分析债券持仓数据，判断是否存在信用下沉特征
- 分析债券信用评级分布、企业债占比、低评级债券占比等

**综合判断**：
- **积极信用下沉**：文本明确提及 + 持仓有明显的低评级债券
- **适度信用下沉**：持仓中有一定的低评级债券，但文本未明确提及
- **高等级为主（信用中性）**：主要持有高等级债券

**降级方案**：关键词匹配（左侧关键词：信用下沉、高收益债；右侧关键词：高等级、AAA）

**基金类型适配**：
- **二级债基**：主要分析债券持仓的信用评级分布
- **可转债基金**：转债信用评级相对不重要，更关注正股质量

---

#### 2. 权益风格分析

**分析方法**：LLM综合分析（文本维度 + 持仓维度）

**文本维度**：
- 使用LLM解析投资策略文本，判断权益投资风格偏好
- 分析文本中关于价值投资、成长投资、均衡配置的描述

**持仓维度**：
- 使用LLM分析股票持仓数据，判断行业配置和风格特征
- 分析行业分布、估值特征、成长性指标等

**综合判断**：
- **价值风格**：低估值、高分红股票占比高
- **成长风格**：高成长、高估值股票占比高
- **均衡风格**：价值和成长股票均衡配置

**降级方案**：基于持仓数据的简单判断（市盈率、市净率、股息率等指标）

---

#### 3. 回撤控制分析

**分析方法**：LLM综合分析（文本维度 + 数据维度）

**文本维度**：
- 使用LLM解析投资策略文本，判断是否有极强的"绝对收益"潜台词
- 分析文本中关于风险控制、回撤管理、防守策略的描述

**数据维度**：
- 计算净值风险指标：最大回撤、年化标准差、卡玛比率等
- 与基金类型的目标回撤范围对比

**综合判断**：
- **回撤控制严格**：文本强调防守 + 实际标准差"低波/中波" + 回撤控制在合理范围
- **风控执行漂移**：文本说严控回撤，但实际标准差"高波"或发生大回撤
- **适度风控**：文本和数据都显示适度的风险控制
- **风控较弱**：文本和数据都显示较弱的风险控制

**降级方案**：关键词匹配（左侧关键词：严控回撤、防守、绝对收益；右侧关键词：进攻、弹性）

**基金类型适配**：
- **二级债基**：目标回撤2%-5%，严控回撤是核心目标
- **可转债基金**：目标回撤5%-15%，接受更大波动换取收益弹性

---

#### 4. 利率波段分析

**分析方法**：LLM综合分析（文本维度 + 持仓维度）

**文本维度**：
- 使用LLM解析投资策略文本，判断是否提及久期波段操作
- 分析文本中关于利率判断、久期管理、波段操作的描述

**持仓维度**：
- 分析利率债（国债/金融债）仓位变化
- 计算重仓债券换手率
- 分析久期变化趋势

**综合判断**：
- **利率波段交易型**：文本明确提到久期波段 + 利率债仓位高/重仓债券多为利率债
- **配置持有型**：文本和持仓都显示长期配置特征，仓位稳定
- **适度参与**：适度参与利率波段操作，但不是主要策略
- **不参与**：基本不参与利率波段操作

**降级方案**：关键词匹配（左侧关键词：久期波段、利率波段；右侧关键词：配置持有、长期持有）

**基金类型适配**：
- **二级债基**：利率波段是重要收益来源，久期管理是关键操作
- **可转债基金**：利率影响相对次要，更关注正股走势和转债估值

---

**LLM分析优势**：
1. **深度语义理解**：能够理解复杂的投资策略描述
2. **上下文关联**：能够结合基金类型特性进行分析
3. **多维度综合**：能够综合文本和数据维度进行判断
4. **灵活适应**：能够适应不同风格的表达方式

**降级机制**：
- 当LLM不可用时，自动降级为关键词匹配或简单判断
- 确保系统在任何情况下都能生成标签
- 降级方案虽然精度较低，但能保证基本功能

**应用场景**：
- 投资理念识别：识别基金经理的投资理念
- 风险偏好评估：评估基金经理的风险承受意愿
- 配置建议：根据基金经理特点提供配置建议
- 基金类型适配：根据基金类型特性提供针对性分析

## 三、数据流程

### 3.1 数据提取流程

```
PDF报告 
  ↓
文本提取 (PDF解析)
  ↓
结构化数据提取
  ├─ 宏观观点提取 (投资策略、宏观展望)
  ├─ 基金经理信息提取 (履历、管理经验)
  ├─ 业绩数据提取 (收益率、波动率、规模)
  └─ 持仓数据提取 (资产配置、股票持仓、债券持仓)
  ↓
数据整合与存储
```

### 3.2 标签生成流程

```
结构化数据
  ↓
标签生成
  ├─ 收益风险标签生成 (基于净值数据)
  ├─ 操作风格标签生成 (基于持仓和宏观观点)
  │   ├─ 短期标签：每个报告期生成一次
  │   └─ 长期标签：聚合所有短期标签
  └─ 个性特征标签生成 (基于持仓和宏观观点)
      ├─ 短期标签：每个报告期生成一次
      └─ 长期标签：聚合所有短期标签
  ↓
画像描述生成
  ↓
标签体系输出
```

### 3.3 短期标签聚合成长期标签

**操作风格标签聚合**：
- **短期标签**：每个报告期生成一次，保存到`short_term_operation_tags/`目录
- **长期标签**：聚合所有短期标签，取出现频率最高的值
- **聚合字段**：投资风格、长期风格、短期风格、风格稳定性、风格演变趋势、风格一致性、风格波动率
- **聚合方法**：使用Counter统计频率，取most_common(1)[0][0]

**个性特征标签聚合**：
- **短期标签**：每个报告期生成一次，保存到`short_term_personality_tags/`目录
- **长期标签**：聚合所有短期标签，取出现频率最高的值
- **聚合字段**：信用下沉、权益风格、回撤控制、利率波段
- **聚合方法**：使用Counter统计频率，取most_common(1)[0][0]

**输出文件**：
- 短期操作风格标签：`output/short_term_operation_tags/{基金名称}_各期操作风格.json`
- 短期个性特征标签：`output/short_term_personality_tags/{基金名称}_各期个性特征.json`
- 长期标签：保存在主标签文件中

## 四、技术实现

### 4.1 数据提取技术

- **PDF解析**：使用PyPDF2提取PDF文本
- **正则表达式**：结构化提取表格和关键信息
- **文本清洗**：处理PDF提取的格式问题
- **基金类型识别**：根据基金名称和投资类型自动识别基金类型

### 4.2 标签生成技术

- **规则引擎**：基于业务规则计算标签
- **统计分析**：基于历史数据统计分析
- **LLM分析**：使用大语言模型分析文本特征
  - 模型：DeepSeek Chat
  - 应用场景：仓位调整风格、操作特征、信用下沉、回撤控制、利率波段等
  - **基金类型增强**：在Prompt中注入基金类型特性信息，提高分析精准度

### 4.3 基金类型区分机制

**核心组件**：
- **基金类型识别器**（`FundTypeIdentifier`）：根据基金名称识别基金类型
- **基金类型特性配置**（`fund_type_characteristics.py`）：定义不同基金类型的特性
- **Prompt增强机制**：在LLM Prompt中注入基金类型上下文

**支持的基金类型**：
- **二级债基**（SECONDARY_BOND）：混合债券型基金(二级)，可投资股票和债券
- **可转债基金**（CONVERTIBLE_BOND）：主要投资可转债，兼具股性和债性

**工作流程**：
```
基金名称
  ↓
基金类型识别器
  ↓
基金类型代码
  ↓
基金类型特性配置
  ↓
Prompt上下文生成
  ↓
增强后的Prompt → LLM分析
```

### 4.4 数据存储

- **JSON格式**：标签数据存储
- **Excel格式**：批量数据导出
- **Markdown格式**：画像描述和报告生成
